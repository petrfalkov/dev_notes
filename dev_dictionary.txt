//error lvls management
Example:
`error_reporting = E_ALL & ~E_NOTICE`
Can be setuped: 
1.	localy in file by function `error_reporting([params]);`
	1. `Without params` - will return current error_reporting lvl on this line
	2. `-1` - will set up all error log output

2. 	As server config in php.ini file `error_reporting = [params]`
